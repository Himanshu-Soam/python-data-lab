## About
This repository contains my **Python Regular Expressions (RegEx)** learning notebook using the built-in **re** module.  
It includes simple explanations and examples to understand and practice RegEx patterns.

## Topics Covered
- Meta characters and special sequences
- Character sets ([])
- Quantifiers (*, +, ?, {m,n})
- Grouping and capturing ()
- **re** functions: match, search, findall, finditer, sub, split
- Match object methods: group, span, start, en
